Fonctionnalités techniques
  • Mode déconnecté :
    • persistance des données : utilisation de $resource, gestion de la mise à jour du cache, HTML5 Web Storage et compression
    • transfert automatique des templates HTML vers un seul fichier JS
    • conversion automatique des medias en base 64
Architecture applicative
  • construite avec Yeoman
  • architecture REST
  • développement d'un module SSO pour serveur CAS (utilisation d'une machine à états finis)
  • déploiement avec GruntJS
    • minification des scripts JS, CSS et des images
    • concaténation des fichiers
  • gestion de version des librairies JS avec Bower
  • execution de tests automatiques avec Karma et Jasmine
UX & UI
  • contribution au module angular-strap
Test
  • écriture de tests unitiares automatiques de non regession pour les services, contrôleurs et directives
  • injection de différents jeux de données
  • écriture de tests fonctionnels
  • gestion des requêtes asynchrones avec $httpBackend
Performance
  • utilisation d'AngularJS Batarang
  • virtual scroll
  • implémentation de ReactJS pour les listes longues
  • utilisation de l'evènement $destroy
  • transclusion
